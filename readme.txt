== Installation ==

1. Upload `divi-custom-list-module` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Does this plugin have any dependency? =

Yes, it depends on the theme Divi by Elegant Themes

= Does this plugin have divi frontend builder support? =

No, we don't support divi frontend builder. We plan to include support in the near future.
